# Magic Voice Agent

Production-ready Flutter architecture for a safe, transparent voice assistant on Android.

## Setup

1. Install Flutter stable.
2. Run `flutter pub get`
3. Run `flutter run`

## Highlights

- Clean Architecture with feature-first modules.
- Voice STT + TTS services.
- AI layer with OpenAI-compatible API and offline fallback.
- Sensitive action confirmation guardrails.
- Command history, repeat command, and theme settings.
